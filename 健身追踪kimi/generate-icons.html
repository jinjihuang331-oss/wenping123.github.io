🏃 健身追踪仪表盘 - 图标生成器

点击按钮生成 PWA 所需的各种尺寸图标

使用说明

  1. 点击"生成所有图标"按钮预览所有尺寸的图标
  2. 点击"下载所有图标"按钮保存所有 PNG 文件
  3. 将下载的图标文件放入项目根目录
  4. 确保文件名与 manifest.json 中配置的一致

需要的图标文件：

  • icon-72x72.png - 小型设备
  • icon-96x96.png - 通用尺寸
  • icon-128x128.png - Chrome Web Store
  • icon-144x144.png - Microsoft
  • icon-152x152.png - iPad
  • icon-192x192.png - Android/Chrome
  • icon-384x384.png - 高分辨率
  • icon-512x512.png - 启动画面
