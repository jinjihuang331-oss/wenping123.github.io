# VR 自然冥想空间

基于 Three.js 的沉浸式 VR 自然冥想体验，支持动态天气、时间切换、音效联动与第一人称漫游。

## 功能特性

### 🌍 动态环境
- **天气系统**：晴天、雨天、雪天、雾天，实时粒子效果
- **时间切换**：黎明、白天、黄昏、夜晚，光照与天空色彩联动
- **场景类型**：森林、湖畔、山谷（UI 已预留）

### 🎮 交互控制
- **第一人称漫游**：WASD 移动 + 鼠标环顾
- **VR 支持**：兼容 WebXR，支持 VR 头显
- **自动漫游**：开启后自动前进，适合冥想观赏
- **速度调节**：0.1x - 3.0x 可调

### 🎵 音效系统
- **程序化音效**：风声、水流声、雨声
- **天气联动**：音效随天气自动切换
- **音量控制**：环境音与音乐独立调节

### 🧘 冥想辅助
- **呼吸引导**：4 秒吸气 → 保持 → 呼气 → 放松循环
- **视觉反馈**：呼吸圆圈动态缩放
- **沉浸 UI**：半透明毛玻璃效果，不干扰体验

## 快速开始

### 本地运行

由于浏览器安全限制，需要通过 HTTP 服务器运行：

```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx serve

# 或使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

### 操作说明

1. **启动体验**：点击屏幕进入第一人称模式
2. **移动**：WASD 键控制方向
3. **环顾**：移动鼠标环顾四周
4. **退出控制**：按 ESC 键释放鼠标
5. **打开菜单**：点击右上角 ☰ 按钮

### VR 模式

1. 连接 VR 头显（支持 Oculus Quest、HTC Vive 等）
2. 点击右下角"进入 VR"按钮
3. 使用 VR 手柄移动和交互

## 技术栈

- **Three.js 0.160**：3D 渲染引擎
- **WebXR**：VR 支持
- **Web Audio API**：程序化音效生成
- **原生 JavaScript**：无框架依赖

## 浏览器兼容性

- Chrome/Edge 90+
- Firefox 88+
- Safari 15+（部分功能受限）
- VR 需要支持 WebXR 的浏览器

## 性能优化建议

- 粒子数量已优化（雨：5000，雪：3000）
- 阴影贴图分辨率：2048x2048
- 雾效距离：50-300 单位
- 建议显卡：GTX 1060 / RX 580 及以上

## 自定义扩展

### 添加新天气类型

在 `app.js` 的 `setWeather()` 函数中添加新的 case：

```javascript
else if (weather === 'storm') {
  // 自定义天气逻辑
}
```

### 调整场景元素

修改树木数量、水面大小等参数：

```javascript
for (let i = 0; i < 80; i++) { // 修改树木数量
  // ...
}
```

### 替换音效

将 `createAmbientSound()` 中的程序化音效替换为音频文件：

```javascript
const audio = new Audio('path/to/sound.mp3');
audio.loop = true;
audio.play();
```

## 已知限制

- 音效为程序化生成，真实感有限（可替换为音频文件）
- 地形为简单正弦波，未使用高度图
- 树木为基础几何体，未使用模型
- 场景切换功能 UI 已预留但未实现
