# 音乐流媒体 UI

一个功能完整的音乐流媒体 Web 应用程序，包含波形可视化、自动完成搜索、时间同步歌词、协作播放列表和离线缓存功能。

## 功能特性

### 🎵 音频播放器
- 使用 **Wavesurfer.js** 实现实时波形可视化
- 播放/暂停、上一首/下一首控制
- 音量控制和静音功能
- 可点击的进度条
- 循环播放模式（关闭/列表/单曲）
- 随机播放

### 🔍 智能搜索
- 实时自动完成建议
- 支持搜索歌曲、艺术家、专辑
- 搜索结果即时播放

### 📜 时间同步歌词
- 自动滚动歌词显示
- 当前歌词高亮
- 平滑的滚动动画

### 👥 协作播放列表
- 模拟多用户协作
- 实时活动日志
- 自动同步歌曲添加

### 💾 离线播放 (Cache API)
- 缓存歌曲供离线播放
- 离线模式切换
- 使用 Service Worker 实现离线支持

### 📱 响应式设计
- 适配桌面和移动设备
- 设备旋转时不中断播放
- 可折叠的侧边栏和面板

## 使用方法

1. 在浏览器中打开 `index.html`
2. 点击"添加歌曲"按钮将歌曲加入播放列表
3. 点击播放按钮开始播放
4. 点击"歌词"按钮显示时间同步歌词
5. 点击云下载图标缓存歌曲供离线播放

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Ctrl + ←` | 上一首 |
| `Ctrl + →` | 下一首 |
| `↑` | 增加音量 |
| `↓` | 降低音量 |
| `M` | 静音切换 |
| `L` | 显示/隐藏歌词 |
| `F` | 全屏模式 |

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **原生 JavaScript** - 应用逻辑
- **Wavesurfer.js** - 音频波形可视化
- **Cache API** - 离线缓存
- **Service Worker** - 离线支持

## 浏览器支持

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 注意事项

- 音频文件使用 SoundHelix 的示例 MP3
- 图片使用 Picsum 随机图片服务
- 在本地测试时建议使用本地服务器以避免 CORS 问题
